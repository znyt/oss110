　　第246章
<br/>　　秦蛮看到几人的穿着，便想起上次暗杀的王秀，必然是和王秀一伙的。他走过去，把二人口中的毒囊摘掉，如此才给二人合上下巴。
<br/>　　“谁派你们来的？”秦蛮。
<br/>　　一群寨主气得哇哇大叫，口中大吼着：“程当家，把他们交给我，我能整的他亲妈都认不出来！”
<br/>　　程大雷压压手，视线落在二人身上。
<br/>　　“不废话，告诉我你们背后的主子，我给你们一个痛快。”
<br/>　　这两人看着周围，其中一人咧嘴笑了：“程大雷，你就要死到临头了，没人能……”
<br/>　　程大雷撇撇嘴，不耐烦的斩掉这人头顶一片血皮，极好的力度控制，头皮被削下，竟没有致命危险，只换来一声渗人骨髓的惨叫。
<br/>　　“我能斩你七刀，不要你的命。”程大雷平静道“我知道你们都是硬骨头的汉子，但我有的是法子把你们骨头磨碎，不要去挑战这一点。”
<br/>　　没有受伤那人终于感觉到一丝寒意，他看着程大雷，保持最后的倔强。
<br/>　　程大雷一剑杵在他眼前，道：“需要我挖出你一只眼睛再说么？”
<br/>　　那人心底一寒，开口道：“是相爷要我们来的，相爷下令，不让你活着走到琴川。”
<br/>　　“姓崔的？”程大雷自然明白他说的是帝国崔相，他奇怪道：“朝廷已肯放过我，姓崔的为何要至我与死地？”
<br/>　　“明玉公主用自己换你一命，但你杀了杨大人，帝国能放你，崔相又怎么可能让你逍遥快活，你这条命早已记在阎王的生死薄上。”
<br/>　　“婉儿？”程大雷又是一愣，下意识道：“这件事和婉儿有什么关系？”
<br/>　　“狗贼，你到现在还不知么，如果不是明玉公主答应嫁到戎族和亲，你又怎么可能活着从天牢出来。”
<br/>　　“是这样么？”
<br/>　　程大雷忽有些失魂落魄，他其实也想过为何自己这次可以脱身，只是没琢磨出什么，还以为是明帝心血来潮，他替明帝守琴川，明帝便放他一条命。
<br/>　　但他不知道，里面还有李婉儿的事。
<br/>　　“程大哥，我们这次要抢亲，抢的就是和亲队伍啊。”黄飞虎笑着道。
<br/>　　胡八刀冷哼一声，气鼓鼓道：“那皇帝老儿忒没有骨气，竟然向戎族和亲，他给戎族当孙子，咱们可不给戎族磕头。”
<br/>　　程大雷离开后一路西行，很少与人打交道，所以他并不知道，在这大半个月时间内，帝国还发生很多事情。
<br/>　　戎族和帝国和谈，要年年纳贡，岁岁称臣。当这件事传开后，在整个帝国掀起轩然大波。
<br/>　　帝国么，总有些大国沙文主义，我们诗书礼乐，锦衣玉食，是高贵的民族，那戎族……不过是一群还茹毛饮血的畜生而已。我们怎么可以向他们称臣，怎么可以向他们投降。
<br/>　　这是帝国之耻，这是每个帝国子民的耻辱。
<br/>　　帝国上下，民怨沸腾，心里都窝着一股羞耻的愤怒。他们再愤怒，当然也不可能拿刀去和戎族拼命，他们怒的是那高高在上的帝国朝廷。
<br/>　　负责和谈的李乐天，王府内天天被人丢进死老鼠，不得不加派一队禁卫军防守。
<br/>　　三千太学生坐在皇宫前，大声诵诗。
<br/>　　『岂曰无衣？与子同袍。王于兴师，修我戈矛。与子同仇
<br/>　　岂曰无衣？与子同泽。王于兴师，修我矛戟。与子偕作』
<br/>　　三千太学生齐诵，要与戎族血战，血战愿为先锋，慷慨激昂，热血沸腾。
<br/>　　明帝处理这件事的方法是很简单的，让禁卫军持大棍驱赶，真有士子被打得头破血流，横尸当场。最后抓了几人，丢进大牢，长安城的大人物把自己家孩子带走，丢进家里关了起来。
<br/>　　如此，这件事算是压了下来，可整个帝国范围内，怒气未消。
<br/>　　“程当家，现在三山五岳，各家山寨都得到消息了，咱们的意思是一样的，皇帝老儿不争气，咱们不能没骨气。”
<br/>　　“把帝国的公主抢回来，不许她嫁到戎族。”
<br/>　　“现在很多江湖人都在筹划这件事，护送和亲队伍的是鱼龙卫，茬子有些硬。咱们峡山想让程当家为首，程当家威望在，把话发出去，绿林道都是肯听的。”
<br/>　　……
<br/>　　后面的话，程大雷其实已听不太清楚，他只感觉脑袋嗡地一下响，胸口疼地厉害。
<br/>　　“她，他们从哪儿走……”程大雷问。
<br/>　　“夕照关啊，出发有一个月了吧，估计再有几天，就要出夕照关了，程当家，咱们要快点行动啊，别被其他人抢了先。”
<br/>　　“秦蛮，备马！”程大雷忽然冷喝。
<br/>　　诸人被程大雷这雷霆声音吓了一跳，秦蛮急忙将一匹快马牵过来，程大雷还剑入鞘，翻身上马。
<br/>　　“大当家，我和你一起去。”秦蛮把斧头递给程大雷。
<br/>　　“不，你们在这里等我，我去去就回。”
<br/>　　程大雷一勒马缰，骏马扬蹄，唿律律发出一声长嘶。
<br/>　　“诸位，夕照关在那个方向？”程大雷马上喝道。
<br/>　　黄飞虎、胡八刀、萧弥峰被程大雷模样吓了一跳，皆不知他要做什么，手却同时指向一个方向。
<br/>　　“在北方。”
<br/>　　程大雷拍马向北。
<br/>　　“程当家这是要干嘛去啊？”诸人都是一头雾水。
<br/>　　“是了，程当家是要单枪匹马去抢亲。”有人恍然大悟。
<br/>　　“真不愧是程当家啊！”诸人又同时发出一声感慨。
<br/>　　……
<br/>　　奔马如飞，踏起黄尘若蛇，前方路上，却突然出现一队人。
<br/>　　“来啦，来啦，他来啦，快做好准备，只有他一个人。”
<br/>　　当程大雷驱马经过时，忽然杀出七八个人，为首一人冷笑，道：“姓程的，天堂有路你不走，地狱无门你自来投，这可是你自己来送死。”
<br/>　　程大雷没有减速，身子突然跃起，身无彩凤双飞翼，接空中爆梨花，还剑入鞘，身子树梢一跃，安安稳稳落在马鞍桥上，继续骑马狂奔。
<br/>　　地上空余几具尸体，绿叶上红色的血滴落，晶莹玉润。
<br/>　　程大雷几乎没有再回头打量一眼，马鞭重重抽下，双眼紧紧盯着前方。
<br/>　　向北，向北，向北。
<br/>　　『我一路向北，离开有你的世界，
<br/>　　你说你好累，已无法再爱上谁』
<br/>
