　　客栈，房内。
<br/>　　萧逸盘膝修炼着。
<br/>　　房内的聚灵阵，功效还是相当不错的。
<br/>　　大量的寒冰灵气，被吸收入体内。
<br/>　　不过，这些寒冰灵气，萧逸并没有将之转化为自身修为。
<br/>　　也就是说，这些寒冰灵气，并没有转化为元气。
<br/>　　而是被萧逸吸入小世界后，强行封存在气泉之外的地方。
<br/>　　整个小世界，最中央是气泉，气泉上方，是武魂。
<br/>　　任何武者，都是这样的情况。
<br/>　　当然了，萧逸有些特殊。
<br/>　　气泉上方，是两个武魂，武魂之下，有一座巍峨冰山，一片沸腾火海，皆由精纯元力凝聚而成。
<br/>　　而在冰山火海与两大武魂之间，是一颗流光溢彩的巨大丹药。
<br/>　　丹药，无时无刻都在受‘火海’之力蒸腾与淬炼，逐渐趋向‘完美’。
<br/>　　哦对了，气泉上，还有一颗金光熠熠的冰火金丹。
<br/>　　而在气泉之外，小世界的一角，正封存着一头巨大的‘鲸鱼’。
<br/>　　正是吞灵冰鲸皇，由精纯灵气而成。
<br/>　　不过，对比之前的庞大，现在已缩水了七八成以上。
<br/>　　这时，萧逸吸收的寒冰之气，皆是在小世界之中。
<br/>　　他不将其转化为自身修为，是不打算突破境界。
<br/>　　现在的他，已经处于地元九重巅峰的临界点。
<br/>　　一个念头，随时可突破天元境。
<br/>　　但，一旦突破，成为天元一重，他的冰山火海，就会悉数消失。
<br/>　　冰山火海，只会在修为境界的临界点出现。
<br/>　　那是由于气泉已十成填满，再无法安放真气或者元力。
<br/>　　加之，因冰鸾剑需要足够凝实而强大的力量，修补剑上那极其微小的裂痕。
<br/>　　由此，而产生。
<br/>　　一旦突破，踏入天元境。
<br/>　　气泉内元力的高度，将只有一成。
<br/>　　自然的，冰山火海将消失掉。
<br/>　　可是，天元境的标志，是液态元力。
<br/>　　进入天元境，萧逸的力量来源，将成了液态元力。
<br/>　　液态元力，远远比不上固态元力的‘冰山火海’。
<br/>　　换句话说，进入天元境，萧逸修为境界是上去了，但实力却不增反降。
<br/>　　‘冰山火海’，才是他的最大战力来源。
<br/>　　无论施展武技，抑或四种强悍火焰。
<br/>　　若是换了在别处，安全的地方，萧逸倒是会考虑突破。
<br/>　　毕竟这样才能更精准地判断自己的境界情况，修为增长，也只是时间问题。
<br/>　　但，这里是危机重重的极寒之地。
<br/>　　现今还与不夜冰宫等大势力结下了仇怨。
<br/>　　他必须保证自己拥有最强战力。
<br/>　　这也是当日第一轮冰海之缘结束后，他没有立刻突破的原因。
<br/>　　事实证明，他的忧虑和谨慎是正确的。
<br/>　　若是之前突破了，如今实力降低，那往后在极寒之地的日子，将变得极其被动。
<br/>　　乃至许多危机，自己都没有把握。
<br/>　　除非能直接突破到天元七重，否则，萧逸暂时不会突破，而是保持着地元九重的修为境界。
<br/>　　天元七重，便是踏入天元境后期。
<br/>　　天元境后期，有着极强的手段。
<br/>　　这一手段，是天元境后期武者拥有极强战力的保证。
<br/>　　若是能掌控这一手段，萧逸才会突破。
<br/>　　事实上，这个时间，不会太久。
<br/>　　萧逸大概有着预估。
<br/>　　那头被封存的吞灵冰鲸皇，虽然已缩水七八成，但其实还是蕴含着相当庞大的力量。
<br/>　　吞灵冰鲸皇，连不夜宫主那等极境强者都觊觎。
<br/>　　可想而知，蕴含力量之恐怖。
<br/>　　若是其他武者吸收。
<br/>　　怕是一大群地元九重武者吸收，都足以跨一个大境界，晋升天元九重。
<br/>　　毕竟能量庞大，且能量层次足够高。
<br/>　　可对他萧逸而言，有着如此庞大的气泉，就算自己独自吸收，也显得有些不够看。
<br/>　　特别是已经被冰鸾剑吸收了七八成。
<br/>　　唯一值得喜悦的，恐怕就是冰鸾剑的那条‘裂痕’，有着些许明显的修补。
<br/>　　而剩下的两三成。
<br/>　　再加上自己一直以来储存的天材地宝与妖兽内丹，炼制成无杂质丹药后。
<br/>　　两者相加，应该够自己升四重左右修为。
<br/>　　另外，就算突破后‘冰山火海’会消失，但里头蕴含的力量，并非是凭空没掉。
<br/>　　还是会有相当一部分，化作液态元力，回归气泉。
<br/>　　这样算来，自己要踏入天元七重，实则并不需要多久。
<br/>　　反正，自己困在地元九重，已经许久了。
<br/>　　还不如继续等下去，等待一个足以厚积薄发的机会。
<br/>　　萧逸的计划，是希望能在此次冰缘大会结束时，回炎武王国前，能突破。
<br/>　　能否成功，便看自己的机缘了。
<br/>　　“冰缘大会第一轮，已有如此收获。”
<br/>　　“第二轮，又该有怎样的精彩呢。”
<br/>　　萧逸自语着，嘴角咧过一道期待的笑意。
<br/>　　冰缘大会，每三十载一次。
<br/>　　每次参加的天骄，个个都收获颇丰，此言非虚。
<br/>　　单单这第一轮，白冰雪四人的修为变化，就是最好的事实。
<br/>　　若非他的气泉太过恐怖，如今也早已修为境界大增了。
<br/>　　时间，渐渐过去。
<br/>　　三天时间，转瞬变过。
<br/>　　今日，便是冰缘大会第二轮了。
<br/>　　萧逸从打坐中醒来，走出房间。
<br/>　　白冰雪等人，也在同一时间出来。
<br/>　　一行五人，往不夜冰宫而去。
<br/>　　如同上次一样，不夜冰宫中，强者云集，各方势力聚首。
<br/>　　在萧逸等人眼中。
<br/>　　那被极光照耀的冰宫，极其绚烂而璀璨。
<br/>　　空中的极光，美丽如旧。
<br/>　　只是，下方的冰宫散发的流光，却已然蒙尘，不再美丽。
<br/>　　道貌岸然，不堪入目。
<br/>　　冰宫执事和弟子，一番记录后，九位冰宫长老现身。
<br/>　　只不过，九位冰宫长老现身后，看了眼萧逸这边，神色有些不自在。
<br/>　　“咳咳。”这时，冰宫大长老故作咳嗽了几声。
<br/>　　随后，宣布了第二轮的规矩。
<br/>　　一番讲解，冰宫大长老高声问道，“可还有疑惑？”
<br/>　　“若无疑惑，冰缘大会第二轮，遗迹之缘，即将开始。”
<br/>　　深厚修为的支撑下，他的声音，传遍全场。
<br/>　　没有人发问，也没有人出言。
<br/>　　冰宫长老，点了点头。
<br/>　　随后转过身，身影一闪，在前方带路。
<br/>　　各大势力武者，紧跟而上。
<br/>　　第二轮的举办之地，同样在不夜城外。
<br/>　　乃是不夜城万里之外的一片庞大雪林中。
<br/>　　这片雪林，极其辽阔，一眼看不到边。
<br/>　　那一棵棵银装素裹的密集树木，千里冰封的地面，甚是美丽、恢宏。
<br/>　　只是，这其中，又蕴含着怎样未知的凶险呢？
