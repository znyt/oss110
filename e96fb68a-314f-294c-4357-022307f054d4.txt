　　第72章
<br/>　　有句话说的好……同行是冤家。
<br/>　　“幸会，幸会。”廖甲双手抱拳。
<br/>　　“好说，好说。”刀疤。
<br/>　　“关门。”廖甲忽然一拍桌子：“抄家伙。”
<br/>　　门咣当一声关上，酒楼的小二厨子都冲了出来。菜刀、板凳、擀面杖都是武器，呼啦一声便将刀疤和青蛇两人围住。
<br/>　　刀疤稍微愣了愣，这哪里是一个流……氓，这明明是一伙流……氓呐。
<br/>　　“爷们，要打便打，嘴里哼一声，算我们犯了怂，打死不用你们埋。可若是打不死，打我们一拳一脚，可都得算银子。”
<br/>　　说着，二人趴在地上，身体蜷缩成一团，用手护住脑袋。
<br/>　　廖甲视线看向苏樱，苏樱冲他轻轻摇头。打，终究是打不起来的，开店也没有把客人打一顿的道理。
<br/>　　“爷们，起来说话吧，事情打算怎么了？”
<br/>　　“三天之内，关门停业，滚出落叶城，若是不答应，咱们走着瞧。”
<br/>　　刀疤二人翻身而起，冲廖甲摇了摇手指，张扬离去。
<br/>　　接下来的事情，便不太好瞧。第二天便有传言传开，说在湖底捞捞出死耗子。又有人说，怪不得那汤味道好呢，原来是耗子肉熬的。
<br/>　　那人问：“为何耗子肉熬得就香呢？”
<br/>　　“那年大旱我吃过老鼠肉，那个味道，美喔。”
<br/>　　这也便罢了，关键是有两辆粪车堵在湖底捞门口，方圆之内，恶臭难闻。
<br/>　　李大头站在二楼往下望，脸上浮现得意的笑：“哼，看现在还有那个人敢上门！”
<br/>　　小二站在一旁，问：“掌柜，粪车堵了他们的门，也相当于变相堵了我们的门，我们现在也没有生意呐。”
<br/>　　“怕什么，等对面关门走人，还愁没有生意。”
<br/>　　如此状况，湖底捞只能大门紧闭，可能恶臭还是一阵阵传进来。苏樱不知自己招惹了谁，对方要用如此下作手段。
<br/>　　终究也不是秘密，隔几天就问了出来，刀疤背后的人，叫做杜茂。
<br/>　　这一天，廖甲快马加鞭回到蛤蟆寨。
<br/>　　“那杜茂叫一个狂嘞，放出话来，三天之内不关门，就要带人砸店。”
<br/>　　“我们兄弟本也不怕他，无非是顾忌老板娘的生意不好做。”
<br/>　　“大当家，咱们一口气憋在心里，可就等着你拿主意呢。”
<br/>　　廖甲委实气得不轻，刚到山寨这番话就咚咚冒了出来。
<br/>　　程大雷正在酿酒，小有进展，酒精度提升不少，只是口感还不太好。
<br/>　　“大当家，带弟兄们杀了他吧，现在老板娘被堵在店里，根本就没办法出门！”
<br/>　　“杜茂……”
<br/>　　程大雷狠狠吐出两个字，一听有人招惹苏樱，顿时气得三尸神暴跳，五灵豪气腾空。
<br/>　　“抬我的大斧来，今晚就入城！”
<br/>　　“好，就等大当家这句话呢。”
<br/>　　“大当家，息怒息怒。”徐神机急忙拦住程大雷：“有道是强龙不压地头蛇，咱们打不得啊。”
<br/>　　“什么地头蛇，我就是地头蛇！”
<br/>　　终究是将暴怒的程大雷拦住，秦蛮，林少羽，黄三元等人聚在一起。
<br/>　　“大当家，你可能不知道杜茂是什么人，那可不是一般人物呐。”徐神机感慨道。
<br/>　　程大雷心里还是有些不痛快：“怎么个不一般……”
<br/>　　“这杜茂曾经酒醉之后，肉拳打死猛虎，如今他在城里养了十几条恶汉，平日里无事时就在一起操练武艺。有道上兄弟投奔他的，无有不收留的，所以他在江湖上威望很高，大家送他一个外号，叫做懒大虫。”
<br/>　　“肉拳打死老虎，难道他叫杜松。”程大雷心底道了一声，问秦蛮：“以你的本事，能徒手打死老虎么？”
<br/>　　“我在山林里也猎过大虫山熊，可那是手中都要家伙，也是十分凶险，若是徒手……我无多少把握。”
<br/>　　程大雷皱紧眉头，秦蛮虽然智商不如何，可在武道上，却是造诣颇深。他说无多少把握，便是无多少把握，难道这杜茂，却是一名能与秦蛮匹敌的强者……
<br/>　　“这懒大虫的名号，我也是听说过的。”黄三元道：“在落叶城里，官府也不敢动杜茂。多少人以结交杜茂为荣，落叶城方圆的大户，只要家里有杜茂的名字，山贼便不会去劫。如今许多大户，每年都会供奉杜松，以求安保。”
<br/>　　“这懒大虫真这么大本事……”程大雷道：“难道我还动不了他？”
<br/>　　“其实大当家也不必烦心，这件事要我说好办得紧。咱们和杜茂是一条线的人，只要大当家的名字递过去，他不会不给蛤蟆寨面子。”
<br/>　　这杜茂了得，可程大雷的名字如今也是传得沸沸扬扬，真要说那个高些，那个低些也是说不太好。
<br/>　　“谁去传话，咱们又不认识杜茂？”
<br/>　　“咱们是不认识，可有人认识呐。”
<br/>　　“谁？”
<br/>　　“高飞虎。”徐神机。
<br/>　　“高飞虎……”程大雷明显愣了一下。
<br/>　　“高飞虎和杜茂的关系极好，二人以兄弟相称，高飞虎每次入城，都是杜茂接待。现在咱们只要托高飞虎的面子，去和杜茂见一面，事情自然可以解决。”
<br/>　　咦，这高飞虎竟然也这么大面子！
<br/>　　程大雷难免吃惊，其实在青牛山，众山寨差不多都是以飞虎寨为首。只不过程大雷对这一点体会不深。
<br/>　　而他和高飞虎的关系……怎么说呢，不是很好。尤其是这段时间，蛤蟆寨将飞虎寨的名头夺得不少。如今有事求到高飞虎头上，他会帮忙么……
<br/>　　“诸位，谁替我去飞虎寨走一遭，请高飞虎帮这个忙。”
<br/>　　最后去飞虎寨的人是徐神机，他见到了高飞虎，说明了情况。令程大雷没有想到的是，高飞虎竟然答应得十分痛快，只说上次程大雷劫了陆哼，替青牛山出了口恶气，帮了程大雷这次，这笔情就算是还清了。
<br/>　　高飞虎甚至派出高飞豹同蛤蟆寨的人一起，到落叶城寻杜茂说情。
<br/>　　程大雷上次大闹黄家露了相，现在满城贴的都是他的描影图形。所以程大雷虽然很愿意会会这杜茂，无奈何却是进不了城。
<br/>　　最后确定下来由徐神机、秦蛮和高飞豹一起进城。
<br/>
