　　女子，似是感受到萧逸冰冷的气息，并未敢再多问。
<br/>　　女子连忙回答道，“火牙盛事，可是我们火牙地域数年一度的大事。”
<br/>　　“由我们火牙地域的武道圣地，火牙宗所举行。”
<br/>　　“每次举行，必是整个火牙地域为之沸腾，各地天骄、武者，纷纷前来参与。”
<br/>　　“甚至于附近地域的天骄和强者也会慕名而来。”
<br/>　　“哦？”萧逸的声音，仍旧嘶哑。
<br/>　　让自己的声线变得嘶哑些，沉重些，并不是什么太难的手段。
<br/>　　“有什么好处吗？”萧逸问道。
<br/>　　“那是当然，不然谁会来？”女子脱口而出，但注意到萧逸的冰冷目光，立刻吓得脑袋一缩。
<br/>　　“好处有二，一是参与盛事的考核过程中过得；二是盛事之后，胜出者能到火牙宗宝库任选三样重宝。”
<br/>　　“而最重要的是。”女子眼中闪过期待之色，道，“最后胜出者，还能得到进入火牙宗修炼，享受不亚于宗内长老亲传弟子的待遇。”
<br/>　　“这才是真正吸引各大天骄和武者的原因。”
<br/>　　萧逸点了点头。
<br/>　　这般听起来，这火牙盛事，似乎也没什么特殊之处。
<br/>　　中域极大，地域无数。
<br/>　　每个地域，都会有属于自己的盛事。
<br/>　　或考核、或比武、或武道参悟等等。
<br/>　　武者的成长，都会在历练、考核等盛事中度过。
<br/>　　盛事，不过是对这些经历的总称。
<br/>　　武者要追求武道，提升武道，自然需要多参加这些盛事。
<br/>　　当然，大部分都是年轻天骄。
<br/>　　而这些盛事，各地域也会不吝举行，不惜付出大量的奖励，吸引地域之内的天骄和武者参与。
<br/>　　其原因，自然是因为这些盛事能让武者们交流，让武者们成长。
<br/>　　这对整个地域来说都是好事。
<br/>　　地域之内的武者，实力越强，这个地域自然就越是闻名。
<br/>　　谁都想成为闻名中域的地域，而不是一个寥寥无名，龟缩一隅的小地域。
<br/>　　自然的，各大地域间，都经常会有盛事举行。
<br/>　　而武者，特别是年轻天骄，则经常游走于这些盛事中。
<br/>　　当然了，某些不世机缘，上古盛事，可是多年难遇。
<br/>　　与剑帝碑盛事、古帝之墓开启等等。
<br/>　　特别是古帝之墓这等有着严格要求的盛事，更是无数武者一生都无法参与。
<br/>　　甚至是顶尖天骄，绝世妖孽，一辈子也就参加那么一次。
<br/>　　“有什么要求吗？”萧逸追问道。
<br/>　　“要求？”女子思索了一下，摇了摇头，“没有。”
<br/>　　“火牙盛事，想来参加，直接来火牙宗即可。”
<br/>　　“前提是别错过时间就行。”女子打趣一声。
<br/>　　萧逸闻言，点了点头，对火牙盛事，大概有了些了解。
<br/>　　这时，萧逸瞥了眼女子，冷哼一声。
<br/>　　心头，则是苦笑一声。
<br/>　　他万万没想到，自己也会遇到‘贼盗’，而且还是个女子。
<br/>　　还好他反应快，拉住了女子，否则乾坤戒就被‘顺走’了。
<br/>　　当然，那乾坤戒内，只是些妖兽尸体，倒没有什么重宝。
<br/>　　真正放置八龙焚火炉、紫电神剑等东西的主要乾坤戒，一直在他手指上，且有禁制存在。
<br/>　　倒是那些放普通东西的乾坤戒，他只是随意带着。
<br/>　　不过，这女子竟然能从他身上‘顺走’东西，倒是厉害。
<br/>　　究其原因，主要是女子的‘空素风狐’武魂，这是一种相当稀有的武魂，位列紫色阶品。
<br/>　　这种武魂，本身作为妖兽，攻击性不强，但却有着极强的御风手段。
<br/>　　而最重要的是，还有着一丝操控空间的本领。
<br/>　　虽然达不到传奇人物‘咫尺天涯’的那等层次，但对于别的普通武者来说，那也是极其棘手的手段了。
<br/>　　这时，女子眼中再次闪过泪花，“前辈，我自幼父母双亡，又终日受人欺凌，都是迫不得已才干这等鼠辈之事。”
<br/>　　“您大人有大量，高抬贵手，放我一马吧。”
<br/>　　萧逸不语，仍旧气息冰冷。
<br/>　　这女子演戏的本事倒是不错，若是不知道她是‘空素风狐’武者拥有者，还真会被她骗过去。
<br/>　　空素风狐，攻击性不强，但极善欺诈。
<br/>　　另外，说到迫不得已，空素风狐武魂拥有者，一般都会有一双巧手，还有极其灵敏的身法。
<br/>　　这女子，凭此无论加入药尊殿还是风刹殿，都能有不少的成就，还不至于要‘偷窃’为生。
<br/>　　而最重要的是，这女子不过23岁左右，却有着圣王境九重的修为。
<br/>　　这等天资，已然相当了不起。
<br/>　　若真如她所说，乃是自幼孤怜，靠自己成长，那她的天资可谓相当厉害了。
<br/>　　就是放到五大学宫中，都是仅次于首席的绝世天骄。
<br/>　　这等人物，终日被欺负？骗鬼吧。
<br/>　　至于让她跟着，主要是想问问火牙盛事的事。
<br/>　　二来，也是自己这身黑袍装扮下，身旁跟着个粗布麻衣的女子，会显得正常些，免得引人注目。
<br/>　　别人不知道也就罢了。
<br/>　　他可是以天地之力的形式感知过，这火牙宗，此刻里头可是藏了数以万计的邪修。
<br/>　　而且其中不乏连他都为之惊惧的黑气气息。
<br/>　　他自是谨慎些。
<br/>　　萧逸暗暗思索着。
<br/>　　一旁女子，也在暗暗腹诽，“哼，老东西，油盐不进，本姑娘总有半分甩掉你的。”
<br/>　　萧逸连火牙盛事这等闻名盛事都不知道，在她眼中，早已将萧逸判断为那等终年闭关，脾气古怪的老怪物了。
<br/>　　……
<br/>　　不多时，二人走到了山峰之上。
<br/>　　山峰之上，火牙宗雄伟的宗门，赫然伫立其中。
<br/>　　二人进入宗门，自有弟子带领。
<br/>　　不多时，二人在带领下，来到了宗门广场处。
<br/>　　随后，弟子离去，准备带领别的参加火牙盛事的武者前来。
<br/>　　萧逸打量了一眼周遭。
<br/>　　整个火牙宗广场，异常辽阔巨大，比一般的宗门里头的正常广场，要大得多。
<br/>　　火牙宗的广场，恐怕占据了这座巍峨高山山峰的一半面积。
<br/>　　一个宗门广场，便占据了宗门一半的范围，这显然是不正常的事。
<br/>　　而宗门广场中间，则有一团熊熊火焰，冲天而起，燃烧不断。
<br/>　　熊熊火海之内，似有一火焰獠牙，瘆人至极。
<br/>　　广场周遭，早已是人山人海。
<br/>　　从各地而来的武者，纷纷惊讶地看着广场中间的那火焰獠牙，面露惧色。
<br/>　　女子注意到周遭武者脸色，看向萧逸，笑道，“前辈，你不必怕。”
<br/>　　“之后火牙盛事的考核，便是在这熊熊火焰之内。”
<br/>　　“若能过，最好；过不了，也就葬身火焰獠牙之处罢了。”
<br/>　　“每次火牙盛事，顶多也就一半死亡率左右，前辈勿怕，勿怕。”
<br/>　　女子轻笑几声，加重了‘死亡’二字。
<br/>　　萧逸闻言，眼眸蓦地一眯。
