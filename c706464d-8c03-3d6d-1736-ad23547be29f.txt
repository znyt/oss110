　　第688章
<br/>　　崔白玉当真对苏樱做过调查，但能查到的资料并不多，因为苏樱并没有多少存在感。有意无意间，崔白玉也将她忽视了，将她当做一个可有可无的人。
<br/>　　直到对面相遇，崔白玉才发现自己犯了错。这个女人底色如何，自己目前还看不清楚，但应不是自己所想的那般简单。
<br/>　　根据调查，苏樱出身不过是普通的商人之家。如这种人，崔白玉平素是见都懒得见的。她们若能和崔白玉说上几句话，扯上一丁点关系，便值得举家欢庆。
<br/>　　可如今毕竟不是从前，现在自己是囚，对方是主。她动一动手指，可以给自己一条活路，也能直接把自己掐死。
<br/>　　“倒是个蛮骄傲的女孩子。”苏樱认真打量着崔白玉：“都说这是名扬长安的美貌，可一个姑娘过分漂亮，于人于己都不见得是什么好事。”
<br/>　　崔白玉深吸一口气，竭力让自己平静下来，像是没有被苏樱的话扰乱心绪。
<br/>　　“程夫人自然很懂做女人的道理，据说程夫人未嫁克夫，再嫁克夫，该是有很多经验。”
<br/>　　对于苏樱的经历，樊梨花也不是很清楚。此刻难免往其身上看了一眼。徐灵儿一惊，对苏樱的底细，崔白玉倒是查得很清楚。她现在担心苏樱会发作，因为崔白玉这话，对一个女人来说可不是什么好话。
<br/>　　苏樱却是表情不变，道：“崔小姐何必言词咄咄，有些事我不会与你争。直说了，我家夫君相中了小姐这幅皮囊，我先过来掌掌眼。我知道崔小姐是从何处来的，自然不会太安份。不过我还是劝你安份些，不然以后便没机会听我讲那些做女人的道理了。”
<br/>　　生死都由苏樱掌握，她自然可以免疫所有言语伤害。而自己争口齿上的上风，是因为自己只能争口齿上的上风。
<br/>　　而这是无意义的。
<br/>　　苏樱的话中又挑明一个事实，自己终将成为程大雷的枕边玩物。她心情自然很糟糕，可这也是没办法逃脱的事实。唯一逃脱的法子，便是引刀成一快，除此之外，她的智慧，她的算计都起不到任何作用。
<br/>　　终究形势比人强。
<br/>　　崔白玉深呼吸，再次深呼吸，想要压抑住纷乱的心绪，想出一个破局的法子。但她没有法子，这才是真正无可奈何之处。
<br/>　　……
<br/>　　此刻程大雷窝在城主府，并不知道苏樱与崔白玉间的小小交锋。他独自窝在书房，琢磨着刚收到的系统提示。
<br/>　　在占领凉州城后，他收到两条奖励以及一道新任务。
<br/>　　奖励包括一副江山图，三次抽奖机会。任务要求他扫平凉州其他势力，一统凉州。
<br/>　　因这条任务并没有时间限制，所以程大雷并不着急去做。而即便系统不发布这道任务，程大雷也是会对浩甲、金城、武威三城动手的。卧榻之侧怎容他人酣睡，只不过，现在局势未稳，自己还没有这个能力罢了。
<br/>　　至于那副江山图，程大雷在书房里翻了半天，也并未找到。最后发现它出现在自己脑海中，只要定神思索，便会缓缓浮现一副地图。
<br/>　　疆域九万里，东临沧海，西靠昆仑，南观群山，北抵草原。帝国十三州一百单八城都被这副江山图囊空，其中凉州三城：蛤蟆城、凉州城、朔方城已经被点亮，其他地区还是混沌不可查看状态。
<br/>　　江山图的功能便是可以查看这三座城的具体情况，人口、商业、军事、农田、特产。
<br/>　　其他倒是不太重要，关键是特产这一项。可以透过江山图，查看某座城池附近有何矿产。如蛤蟆城有铁块，朔方城有种草药……现在程大雷没精力开矿，可以暂且放一放，等局势稳定下来再说。
<br/>　　至于三次抽奖次数，程大雷暂时感觉用处不大，也可以同样放一放。现在自己尚未暂稳脚跟，万一给自己搞出个传说级别的敌人，那真是自己把自己玩死。
<br/>　　目前凉州城渐渐步入正轨，正在复苏。蛤蟆城不会有太多发展前途，维持现状便可以。至于朔方城，招募人口的工作还在进行，但距离程大雷完成任务要求的十万人口，还是有一定差距。
<br/>　　在最初的忙乱之后，现在的凉州城已渐渐稳定下来。程大雷也不像刚开始那样忙碌，他现在已觉得没太大的事，局势会越来越好的。
<br/>　　程大雷并不知道，当他这么想的时候，帝国许多人的心情都糟糕透了，而且会越来越糟糕。
<br/>　　长安，未央宫。
<br/>　　李乐天的手重重敲在龙案上，脸上肌肉绷紧，怒目圆睁，很显然已经怒到极致。
<br/>　　能让帝国天子如此愤怒，程大雷要负主要责任。程大雷拿下凉州城，好死不死，竟还派人给长安天子递了一封信。当然，也可以说是奏折。信上说，自己是奉了明帝遗诏，执匹夫剑斩天下不臣。这宋伯康不是个东西呐，密谋造反，居心叵测，所以我便将他赶跑了。
<br/>　　这样的话明显就是欺负李乐天是个傻瓜，程大雷还恬着脸向李乐天讨个封赏。若是帝国强盛时，李乐天恨不得御驾亲征，将程大雷扒皮碎骨，方泄自己的心头之恨。
<br/>　　然而，现在帝国不够强大。
<br/>　　李乐天目前面临的最大问题，绝非程大雷，而是那在相府深居简出的老人。
<br/>　　李乐天虽然是天子，但权力实在有限。他其实也明白，掌握军权便掌握一切，可三十万禁卫军中，多是相府门生。原先尉迟离在的时候，将军府还能和相府掰掰手腕，可尉迟离一死，相府便开始用自己的人代替将军府的人。原先忠于将军府的世家豪门纷纷改弦易张。
<br/>　　所以李乐天能动用的力量当真不多。
<br/>　　仅仅军中是相府的人么？就算在这大内皇宫，也不知多少宫女太监是相府安插的眼线，监视自己的一举一动。
<br/>　　自己哪怕某天多吃了几碗饭，都有人立刻禀报给相府知道。
<br/>　　他坐在龙书案后，心情实在糟糕到极点，嘴里呐呐叹道：
<br/>　　“孤家寡人，孤家寡人，朕当真成了孤家寡人。”
<br/>(本章完)
