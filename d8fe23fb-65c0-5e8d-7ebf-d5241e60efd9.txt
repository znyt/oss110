　　“前行数十里？”
<br/>　　萧逸皱了皱眉。
<br/>　　这第二层的尽头，一望无际，远在千里以外。
<br/>　　以这二人的实力，竟也才前行数十里？
<br/>　　桌万宗，自不必多说，已觉醒第二次武魂，且有圣皇境八重修为。
<br/>　　他乃气宗第一天骄，实力之强自不必多说。
<br/>　　更兼还能凝聚气剑力。
<br/>　　此人的实力，若放到外面的其它地域，怕是圣皇境巅峰强者也未必是他对手。
<br/>　　子风师弟，则更强。
<br/>　　林子风作为林家少家主，武魂第二次觉醒之事，自也不存在难度。
<br/>　　武道圣丹，家族强者的牵引，大量修炼之物等等，丝毫不是问题。
<br/>　　此时的林子风，乃是圣皇境九重修为。
<br/>　　算上魂剑力的增幅，真正实力，绝对达到武道大能层次。
<br/>　　二人联手之下，竟也对这万剑之阵毫无办法？
<br/>　　“实力尽出了？”萧逸看向林子风，问道。
<br/>　　林子风点了点头，又摇了摇头，“倒是还有些许手段未出。”
<br/>　　“可有把握？”萧逸问道。
<br/>　　林子风挠了挠头，“把握还是有的，虽然不是十足把握。”
<br/>　　“可需要我帮你？”萧逸轻笑道。
<br/>　　“帮？拿什么帮？”桌万宗抢先道，“本公子话先说在前面，我没兴趣与你联手。”
<br/>　　萧逸瞥了桌万宗一眼，不屑道，“我也没有兴趣，且也未问你。”
<br/>　　萧逸看向林子风。
<br/>　　林子风摇了摇头，“虽然我知晓萧逸师兄本事过人。”
<br/>　　“不过这是我的考验之路，我想靠自己走到第二层尽头。”
<br/>　　“好，那便随你。”萧逸点点头，拍了拍林子风的肩膀。
<br/>　　“那我先行一步了，第二层尽头见。”
<br/>　　“嗯。”林子风认真地点了点头。
<br/>　　一旁，桌万宗嗤笑一声，“我和子风二人都寸步难行，就凭你一人？笑话罢了。”
<br/>　　萧逸未理会桌万宗，身影一闪，直接前行。
<br/>　　身影刚入剑阵之内，一把利剑，已疾速袭来。
<br/>　　利剑的破空声，异常激烈。
<br/>　　不愧是那些剑碑主人曾经仗之横行天下的利剑，果真不凡。
<br/>　　不过，在这里的毕竟只是一把剑罢了，而并非剑碑主人持之。
<br/>　　故萧逸并不惧之。
<br/>　　嗖…萧逸指尖一道剑气打出，瞬间拦下了这把利剑。
<br/>　　“圣皇境巅峰水准。”萧逸只看了一眼，便感知出利剑袭来的威力。
<br/>　　同时，利剑被拦下，一道氤氲光芒，激射而来。
<br/>　　萧逸不躲不闪。
<br/>　　他明显感知到这道光芒没有丝毫威力。
<br/>　　果然，光芒打到他身上后，他身上的气息陡然增强了一分。
<br/>　　“原来如此。”萧逸面露喜色。
<br/>　　对比第一层空间，这第二层空间的好处，多得多了。
<br/>　　嗖…嗖…嗖…
<br/>　　萧逸的身影，不断闪烁。
<br/>　　每前行一里距离，便会多一把利剑朝他袭来。
<br/>　　而每挡下一把利剑，就会得到一份元力的馈赠。
<br/>　　这些元力，打入他体内后，霎时化作他本身的精纯元力。
<br/>　　“数万神兵利器，数万份层次不低的元力。”萧逸脸上惊喜无比。
<br/>　　若是他能一直走到第二层尽头，挡下所有利剑的攻击，便能得到第二层内所有神兵利器的元力馈赠。
<br/>　　这般庞大力量，或许足够他直接升到圣皇境了。
<br/>　　嗖…嗖…嗖…
<br/>　　待得萧逸前行至数十里距离时，已是数十把利剑袭来。
<br/>　　若单纯袭来也就罢了，可别忘了，这里可是剑阵之内。
<br/>　　也就是说，数十把利剑，乃是自成剑阵攻来。
<br/>　　这绝不是单纯的数量增幅。
<br/>　　威力更是直线飙升。
<br/>　　嗖…萧逸一道星光剑气打出。
<br/>　　“达到武道大能水准了。”萧逸感知出数十把利剑的威力。
<br/>　　嗖…萧逸身影一闪，后退了几步。
<br/>　　这里距离第二层尽头，还有很长的距离。
<br/>　　故萧逸打算这些袭来的利剑，若能躲避，便直接躲避了。
<br/>　　也好省些力，对付后面距离更长后威力更强的利剑。
<br/>　　只不过，他才刚退了几步，虽躲过了袭来的利剑，身上的气息，却陡然变弱。
<br/>　　“嗯？”萧逸脸色微变。
<br/>　　他明显感知到，自己后退的一瞬间，身上原本得到的元力，瞬间便自动消散了些许。
<br/>　　如果他没有猜错的话，从这些神兵利器上得到的一份份元力，只是暂时的。
<br/>　　若他能彻底通过这个剑阵，走到第二层尽头，这些庞大元力便彻底化作自身修为。
<br/>　　若是中途不敌，乃至退回原地，得到的元力，便会悉数消散。
<br/>　　也就是说。
<br/>　　这第二层空间的考验，若过了，自是得了一份天大的好处。
<br/>　　若过不了，便算白了，啥都得不到。
<br/>　　“哼。”萧逸冷哼一声，身影猛地前行。
<br/>　　不过片刻，身影已前行百里之外。
<br/>　　同时，向他攻来的利剑，已达百把。
<br/>　　百把利剑，自成剑阵，玄妙围攻而来。
<br/>　　越是前行，利剑越多，激发的剑阵威力便越强。
<br/>　　外头原地，林子风满脸喜色，“不愧是萧逸师兄，轻松便横跨百里。”
<br/>　　“哼。”桌万宗脸上闪过一丝不悦。
<br/>　　“子风，你可是在长他人志气灭自己威风？”
<br/>　　林子风耸耸肩，“在子风心中，萧逸师兄不是外人。”
<br/>　　“你…”桌万宗脸色一冷，“你这般意思，可是萧逸这小贼不是外人，我桌万宗才是？”
<br/>　　“你别忘了，我们才是剑域剑者。”
<br/>　　“你林家与我气宗，更是交好无数年。”
<br/>　　“林家归林家，我林子风归林子风。”林子风摇了摇头，并未再多言。
<br/>　　桌万宗双眼一眯，看着远方不断前行的萧逸，蓦地一笑。
<br/>　　林子风瞬间注意到桌万宗的笑意，皱眉道，“你想做什么？”
<br/>　　“你说呢？”桌万宗冷笑一声，“既然萧逸这小贼自己在前面打头阵，我们现在前行，难度便会大减。”
<br/>　　“没用的。”林子风嗤笑一声，“剑帝碑内的手段，没你想象的那么简单。”
<br/>　　“现在才百把利剑，还好说。”
<br/>　　“待得后面动辄千把利剑时，即便萧逸师兄承受了大部分的攻击和压力，也不是你现今的修为能扛过去的。”
<br/>　　“若是加上这个呢？”桌万宗冷笑一声，指尖一道剑气凝聚而出。
<br/>　　剑气，极强。
<br/>　　林子风脸色一变，这道剑气的强度，仅比他感受过的他父亲林霄的剑气，若是些许。
<br/>　　整个气宗，能凝聚出这道剑气的，只有一人。
<br/>　　“是气宗宗主的剑气？”林子风眉头一皱。
<br/>　　“哈哈哈哈。”桌万宗得意大笑，“第二层尽头的宝贝，归我了。”
<br/>　　话音落下，桌万宗瞬间闪身前行。
