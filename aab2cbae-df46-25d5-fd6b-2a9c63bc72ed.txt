　　第39章
<br/>　　“这……男男授受不亲，阁下请自重。”程大雷将李行哉的手指掰开。
<br/>　　李行哉依旧目光如火：“先生大才，小小青牛山怎么容得下，自该有一更广阔的天下，容先生写下名字，青史留名。”
<br/>　　“呦，我的名字不过三个字，占不了那么大地方。”程大雷笑笑，忽然问：“你有门路……什么门路？”
<br/>　　李行哉语塞，略沉默片刻：“自有容先生施展才华的门路。”
<br/>　　“罢了，我胸无大志，既不想给人扛刀，也不愿为人站岗。”程大雷站起身来：“趁着天还不晚，我去看看辣椒磨得怎么样了。”
<br/>　　说着，程大雷便向屋外走去，走了一半，忽地扭过头来：“你的剑不错，要不给我留下？”
<br/>　　李行哉目光忽然一厉，眼里像是藏着刀，随之刀芒消散，他又换上温和的表情。仔细打量着程大雷，似乎想要看出些什么，可程大雷就那样笑着，好像不过随口一句无心之言，笑着走出屋子。
<br/>　　院外响起几个人推磨的声音，在忙碌着把辣椒磨成粉末。李行哉在蛤蟆寨住了十几日，也吃了几顿辣椒做的菜，那种痛快淋漓的味道，就连从小锦衣玉食的他也从未体味过。程大雷说那种红色的果子叫做辣椒，以前倒是没有见过。
<br/>　　目光落在桌上，那里放着一柄剑。黑色剑鞘，是用上好的犀牛皮鞣成的，剑柄上挂着一缕流苏，这可以方便握剑，吸收掌心的汗水。除此之外，不过是朴素的一柄剑，有人喜欢在剑上加各种宝石，金饰，银饰……李行哉是不喜欢的。
<br/>　　在剑颚的位置，有一个小小的『陆』字，若是不仔细看，根本不会察觉。
<br/>　　“想要我的剑么？”李行哉轻轻呢呐：“口气不小啊。”
<br/>　　……
<br/>　　三天无非一弹指。
<br/>　　这天是个晴天，赤白的日头高悬，空气静得没有一丝风，树梢的叶子都是动也不动一下。
<br/>　　哒哒的马蹄声响起，从很远的路上，行过来一队人马。队伍约有百人上下，前后是十人伍的骑兵，中间是佩刀的步兵。队伍保护着中间的马车，只从外面看不见马车上装着些什么。
<br/>　　队伍最前方行着的一人，黑衣黑马，马上一柄镔铁大棍。
<br/>　　“陆将！”
<br/>　　一名斥候快马从青牛山方向奔过来，勒马在陆哼面前停下。
<br/>　　“禀陆将，前面就是青牛山，伢子已经探过，路上没有埋伏，青牛山各寨的山寨都在山下等候。”
<br/>　　有时候一两名斥候就可以解决很多问题，如果韩玄之还在，就应该知道，在通往名将的道路上，他连最基本的素质都没有。
<br/>　　陆哼抬首，他的肤色是一种经年经历阳光暴晒的枯黄，走在这样炎热的天气，虽然脸上热汗滚落，却也并没有任何不适。
<br/>　　“再探！”
<br/>　　“是！”
<br/>　　斥候呼啸一声，立刻调转马头，向来的方向奔去，于此同时，在青牛山的另一头，一名斥候正快马向这边奔来。
<br/>　　两名斥候一来一去，循环禀报，沿途的状况会在最短时间传到陆哼耳中，若有埋伏，陆哼所率领的一百精兵将在最短时间内趟平道路。
<br/>　　而今天的青牛山下，并无埋伏。
<br/>　　飞虎寨高飞虎高飞豹兄弟，杏花岭小白狼，秃头坡熊大熊二兄弟……几乎青牛山所有山贼都站在各自的山头下，隔三五里就可以看到一撮山贼，但他们之所以出现在这里，并不是为了打劫，而是为了……迎接。
<br/>　　这条杀鸡儆猴的计策，不知是出自陆哼还是出自薛半川，但不管是谁，却都非常有效。
<br/>　　用雷霆手段血屠朱家寨，将朱家寨上下三十几口全部砍掉脑袋，这自然是很残忍的，但残忍有时候也是一种显示实力的手段。这让青牛山的山贼看到薛半川的实力，青牛山这帮乌合之众根本不是对方的敌人，只要有人敢打这批寿礼的注意，就要有心理准备等来灭寨之祸。
<br/>　　所以，各家酝酿打劫寿礼的主意也就烟消云散，而且还要在陆哼经过的时候，在山下迎接。
<br/>　　这自然是一件很屈辱的事情，众山贼也并不缺乏穷凶极恶的亡命徒，可是在绝对的实力面前，凶狠并无任何意义。
<br/>　　都是活着，凶狠是活下去的法子，卑微的俯下身子也是，在凶狠没办法让人活下去的时候，没人介意卑贱的低头。
<br/>　　这个时候，陆哼的队伍已经行进青牛山。
<br/>　　“来了，来了！”高飞豹低声道。
<br/>　　青牛山各山寨几乎同一时间得到这消息，人人心里都提了一口气，站直身子，头却是微微低了低。
<br/>　　“大当家，打探清楚了，青牛山五十二家山寨差不多都在山下候着。”于虬然道。
<br/>　　“这就好。”高飞虎点点头，所有人一起丢人，就显得自己不那么丢人了，他忽然一楞：“差不多是什么意思？”
<br/>　　“蛤蟆寨寨门紧闭，没有人在山下。”于虬然。
<br/>　　高飞虎猛地抬头，愣愣道：“他究竟想干什么！”
<br/>　　朱家寨被灭，其实每个人心里都憋着一口气，这感觉并不令人愉快。可不愉快又能怎样，大家都不愉快，可不也同样憋着气，忍屈受辱的在山下守着。
<br/>　　这口气，大家都忍得下，你凭什么忍不下。为这种事生气，紧闭寨门，如果因此触怒陆哼，他灭的了朱家寨，还灭不了你蛤蟆寨。
<br/>　　年轻人呐，还是太幼稚。
<br/>　　高飞虎心里想着，派一个喽啰去蛤蟆寨，让程大雷也老老实实把寨门打开，在山下迎接。
<br/>　　可就在这个时候，陆哼的队伍已经行到高飞虎面前。他坐在马上，遮住天上的日头，阴影投在高飞虎身上。
<br/>　　高飞虎抬起头仰视着马上的陆哼，仰视总会给人带来一种屈辱感，如同趴着的奴隶。
<br/>　　“你带刀干什么？”陆哼问，沉重的铁棍举起，搁在高飞虎肩上。
<br/>　　多么屈辱啊，铁打的汉子像一个娘们一样被人握在手上。
<br/>　　飞虎寨有一个是一个，脸上肌肉跳动，胸腔内酝酿着怒气。只要高飞虎喊一声，大家不介意一拥而上，哪怕是鸡蛋撞石头。
<br/>　　高飞虎扣紧牙关，慢慢低头，将大刀放在地上，
<br/>　　“放下武器，送陆将军过山！”
<br/>　　陆哼收回铁棍，咧嘴一笑。
<br/>　　“好，那你就在后面跟着吧。”
<br/>　　
<br/>(本章完)
