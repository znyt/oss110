　　“桀桀桀！”
<br/>　　血无影见柳云飞乖乖让开，顿时发出一道得意的邪笑，伸手在灵韵那美丽的下巴下一挑，
<br/>　　“美人儿，乖乖跟本少走吧！”
<br/>　　灵韵顿时浑身巨颤，可是面对血无影，她根本没有反抗的勇气，不由得转头望了一眼灵山，希望自己的父亲可以拼死维护自己。
<br/>　　然而，
<br/>　　灵山却重重叹息一声，直接将头扭到一旁，
<br/>　　“女儿，为父对不起你……”
<br/>　　听到这话，灵韵心中最后的一丝希望，也彻底湮灭！
<br/>　　自己仰慕的那个男人袖手旁观，连自己的父亲也选择视而不见，她还能指望谁来救她？
<br/>　　就在灵韵心如死灰，准备跟随血无影离开的时候，忽然后方响起一道清脆的声音，
<br/>　　“我不许你欺负我姐姐！！”
<br/>　　伴随着这道动听的声音响起，一道纤瘦的少女身影，从后方人群中冲了出来，
<br/>　　这人不是旁人，自然正是灵溪！
<br/>　　灵韵看到自己妹妹站了出来，先是心头一暖，继而脸色大变，
<br/>　　“灵溪！这里没你的事，你快回去！！”
<br/>　　这个血无影，就是一个不折不扣的淫棍，万一这个禽兽，将主意打到自己妹妹的身上，那她就算死了，也不会瞑目的。
<br/>　　一念及此，灵韵一把拦住灵溪，便要将她往人群后面去推。
<br/>　　然而，
<br/>　　她最担心的事情，最终还是发生了！
<br/>　　“等等！”
<br/>　　血无影突然开口，同时手掌虚抓一拉，
<br/>　　刺啦啦！
<br/>　　灵韵和灵溪，便直接被他从人群中拉了回去。
<br/>　　“啧啧！没想到你们还藏着这么一个小美人！好一对姐妹花啊！”
<br/>　　灵韵顿时大急，连忙将灵溪护在身后，
<br/>　　“血少！你有什么就冲我来！我妹妹她还小，不懂事，求您，求您就放过她吧！”
<br/>　　就连灵山也坐不住了，不由得上前一步，哀求道：
<br/>　　“血少侠，求您放过小女！”
<br/>　　“滚！”
<br/>　　血无影连看都没有看灵山一眼，直接随手一掌拍出，
<br/>　　轰！
<br/>　　灵山直接被震飞了出去，狂喷了一大口鲜血，显然已经受了不轻的伤。
<br/>　　“父亲！！”
<br/>　　灵韵和灵溪两姐妹，顿时纷纷齐呼了一声，奈何身体被血无影的真元禁锢着，根本动弹不得。
<br/>　　其余黑岩城的众高手，看到城主落得如此惨状，顿时一个吓得纷纷倒退。
<br/>　　就连柳云飞，原本想开口求情，看到这一幕之后，也不由得将到了嘴边的话，又生生给咽了回去，
<br/>　　“本少看上的女人，我看哪个再敢废话？”
<br/>　　说着，血无影便一脸婬光的朝灵溪抓去，后者顿时吓得俏脸惨白。
<br/>　　然而就在这时，
<br/>　　一道淡漠的声音，陡然自人群中响起，
<br/>　　“血无影是吧？你若是敢碰灵溪一下，你的这条胳膊，就不必要了！！”
<br/>　　轰！
<br/>　　这道声音一出口，全场众人顿时一片轰然，
<br/>　　连柳云飞和灵山都不敢硬撼血无影的锋芒，何人如此大胆，竟然敢说出这种话来？
<br/>　　众人不由得纷纷循声望去，
<br/>　　只见那说话之人，是一名穿着有些周围格格不入的瘦削少年。
<br/>　　此人自热正是叶尘！
<br/>　　旁人的死活，叶尘才懒得理会，不过如今牵扯到了灵溪小丫头，他就不能再不管了。
<br/>　　“是他？”
<br/>　　灵韵原本心头闪过一抹惊喜，等看到是叶尘之后，顿时又一阵失望。
<br/>　　在她看来，叶尘的修为，连他都有所不如，更遑论和血无影相比了！
<br/>　　虽然勇气可嘉，却也不过是白白送了自己的性命而已。
<br/>　　血无影在叶尘身上扫了一眼，顿时阴阴一笑，
<br/>　　“小子！你刚刚说什么？可敢再说一遍？”
<br/>　　哗啦啦！
<br/>　　叶尘周围，那些黑岩城的高手们，顿时慌忙纷纷四散开来，生怕离的太近，被他连累。
<br/>　　叶尘连眼皮都没有抬，双目低垂，望着地上，再次淡淡开口道：
<br/>　　“好吧，我收回刚才的话……”
<br/>　　众人还以为叶尘怂了，顿时纷纷忍不住嗤笑起来。
<br/>　　这家伙是来搞笑的吗？
<br/>　　他以为血无影是这么好招惹的？
<br/>　　血无影也顿时笑了，
<br/>　　他刚刚还在好奇，在巨门星域，居然还有人敢和自己这么说话？
<br/>　　果然这么快，就臣服在自己的王八之气下了，这才是蝼蚁应有的姿态嘛。
<br/>　　血无影心里正得意着呢，叶尘却又开口了，
<br/>　　“给你十息时间，不滚，就死！”
<br/>　　轰！
<br/>　　叶尘这话一出口，众人全都傻了，纷纷两眼瞪得老大，朝叶尘望去，一个个犹如看傻子一般。
<br/>　　“我没有听错吧？这小子竟然让凶名赫赫的血公子滚？”
<br/>　　“他是得了失心疯吗？”
<br/>　　“就算是想死也不用这样吧？”
<br/>　　……
<br/>　　众人议论纷纷，血无影更是气得怒极反笑，眼眸中透着浓浓的杀机！
<br/>　　“好好好！没想到小小的黑岩城之中，竟然还有如此胆量之人！很好啊！”
<br/>　　黑岩城的众高手们，听到这话，顿时急得不行，
<br/>　　“血公子莫要误会！这人可并非是我黑岩城之人呐！”
<br/>　　以他们对血无影的了解，此人残暴无比，若是叶尘这话惹得他大怒，说不定还会牵连到他们。
<br/>　　“哼！”
<br/>　　血无影冷哼一声，
<br/>　　“看来本少今日若是不给你们一个深刻的教训，恐怕要不了多久，就没人把本少放在眼里了！”
<br/>　　轰！
<br/>　　说完这话的同时，血无影身上的气势，猛然暴涨起来，透着浓浓的杀意！
<br/>　　一时间，所有人的心，全都狠狠一颤，尤其是黑岩城众人，更是个个脸色惨白。
<br/>　　“完了完了！看来血公子是真的动怒了！”
<br/>　　“纵然我们这些人联手，或许可以抵挡，可是血公子的背后，可是血月宗啊！”
<br/>　　“都怪这小子！他自己作死，却连累我们黑岩城，都要跟着他一起倒霉了！”
<br/>　　……
<br/>　　黑岩城众人纷纷朝叶尘怒目而视，都恨不得他赶紧去死。
<br/>　　在他们看来，如果不是叶尘作死，惹得血无影大怒，他们最多也就损失两个女人而已。
<br/>　　“不知死活的小子！本少就先拿你开刀！”
<br/>　　血无影暴喝一声，直接朝叶尘虚手一抓，
<br/>　　轰！
<br/>　　一道巨大的金色手掌虚影，凭空出现在九天之上，携带着惊天威能，在众人的惊愕目光中，朝叶尘笼罩而去。
