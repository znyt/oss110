　　四长老感受了一番自己体内被压制的真气，顿时暴喝一声，“黑墨曼陀罗，果然是慕容墨的武魂力量。”
<br/>　　“黑墨曼陀罗？”萧逸听到四长老的话，再看了眼体内漆黑如墨的那朵花骨，顿时想了起来，这黑墨曼陀罗，在大长老的炼药心得中曾经提起过。
<br/>　　黑墨曼陀罗，三品天材地宝之一。
<br/>　　而在武魂中，这是黄阶巅峰武魂，比之萧若寒的火云蟒还要强悍得多。武魂力量含有剧毒，攻击时发出毒气弥漫周身。
<br/>　　就算是实力相当的武者，也会因这些毒气而降低修为，导致不敌。甚至，若接触这些毒气太久，武者直接当场殒命，骨头腐烂，凄惨而死。
<br/>　　这是一种非常恶毒的武魂，由多年前慕容家大长老慕容墨觉醒而拥有。
<br/>　　如今萧家矿脉内众人危矣，萧逸也是心头着急，但也只能稳住心神，快速思索办法。
<br/>　　“两个先天武者，就算我仍有战力，也无法对付。为今之计，只有让四、六长老恢复战力，方可对付他们。”萧逸心头暗暗想着。
<br/>　　萧逸暗暗向四六长老打了个眼色，示意他们拖延时间。
<br/>　　四六长老会意，虽然不明白萧逸为什么这样做，但还是照办。
<br/>　　这时，慕容槐冷笑一声，“萧家四长老，果然有本事。不错，这乃是我们慕容家大长老的黑墨曼陀罗。”
<br/>　　“哼。”慕容蝎冷哼道，“这只是大长老事先凝聚出来并交给我们的武魂力量，这才只是暂时压制你们的真气数个小时。”
<br/>　　“若是大长老亲至，你觉得你们现在还有说话的机会吗？早就化为一滩血水了。”
<br/>　　慕容槐道，“不必与他们废话了，先杀了他们再说。”
<br/>　　两人刚要动手，四长老大喝一声，“老五，你就任由他们杀我们？”
<br/>　　一声‘老五’，让五长老的身体一震，目光有些复杂。
<br/>　　四长老沉声道，“老五，忘了我们当年如何出生入死了吗？忘了那些一起守护萧家，大杀四方的峥嵘岁月了吗？你真要帮慕容家的小人对付我们？”
<br/>　　五长老也是先天武者，而且实力极强，否则也不会在萧家拉拢到七八九三位长老。若他能倒戈相向，萧家这次的危机将会减缓许多。
<br/>　　慕容槐见五长老神情挣扎，当即道，“五长老，你忘了这群人在萧家是如何为了萧逸挤兑你，为难你吗？你忘了他们对你儿子的不公平吗？”
<br/>　　“你放屁。”六长老大骂道，“老五，这么些年来，在萧家我们如何待你，你清楚得很。你在萧家暗地里做的那些事，你以为我们一个个长老都是瞎子？都不知道？只不过是不想让你难堪罢了。你勿要听信慕容家小人的恶毒之言。”
<br/>　　五长老的脸色挣扎得更加厉害。
<br/>　　四长老见状，当即道，“老五，如今我们萧家所有出色的子弟都在矿脉中，若让慕容家的人得逞，那些子弟危矣。日后，萧家将一蹶不振，你想看到萧家破败没落？”
<br/>　　五长老竟握了握拳头，似乎想要做什么。
<br/>　　慕容槐连忙大喝一声，“五长老，切勿中计。”
<br/>　　“只要萧逸死了，萧家再无子弟能在天赋上超越你儿子。只要四、六长老死了，那三长老没了支持，也比不过你。你的家主之位，指日可待。”
<br/>　　慕容槐攻于心计，一下子就刺中了五长老的死穴。
<br/>　　听到‘家主之位’四个字。
<br/>　　五长老脸上的挣扎之色全无，拳头也放松了下来。
<br/>　　他对萧家加之之位觊觎了这么多年，绝不可能放弃。
<br/>　　慕容槐阴笑一声，看向一旁的萧若寒，道，“若寒，你现在肯定很想念娇儿吧。”
<br/>　　“这是当然。”一旁的萧若寒点点头，满脸幸福。
<br/>　　慕容槐道，“等你父亲成了萧家家主，你便是少家主；娇儿她是我们家的千金，慕容家当代家主的女儿；到时候，你与她门当户对，便直接把婚事结了。”
<br/>　　“如此甚好，谢过慕容槐长老了。”萧若寒甚至对慕容槐行了一礼。
<br/>　　慕容槐点点头，看向五长老，道，“五长老，你也想看到我们两家结秦晋之好，称霸这紫云城的吧。”
<br/>　　五长老点点头，而后看向四六长老，道，“老四、老六，你们安心去吧。等我当上萧家家主，萧家只会比现在更好，而不会破败没落。”
<br/>　　“你...”四长老顿时气急，“老五，你真的没救了。”
<br/>　　慕容槐大喝道，“五长老，还和他们废什么话，快杀了他们。”
<br/>　　“好。”五长老面露杀意，立刻出手。
<br/>　　“不好。”一旁看着的萧逸心头一惊，“遭了，还差一点儿时间，只能跟他们拼了...”
<br/>　　萧逸心中刚做着最坏的打算....
<br/>　　孰料，场面忽然变化，甚至让所有人反应不过来。
<br/>　　五长老冷冷地看着四六长老，扬起了刀子，狞笑道，“老四、老六，我会给你们个痛快的，我...”
<br/>　　他的话还未说完，一把锋利的刀子忽然从他背后刺入，从胸膛刺出。
<br/>　　“噗。”五长老吐出一口腥血，不可置信地转过头，看着慕容蝎，问道，“你...你做什么？”
<br/>　　这时，慕容槐竟也忽然偷袭出手，全力一掌打向他。
<br/>　　五长老受重伤，刚要躲避，立刻被慕容蝎牵制。而后，慕容槐的全力一掌，直接震碎了他的心脉。
<br/>　　“你们...”五长老嘴里噙着血水，满脸不可置信。
<br/>　　慕容槐阴鸷地笑道，“五长老，要不是你对我们放下戒心，我们还真的难以杀你阿。”
<br/>　　慕容蝎更是讥讽道，“五长老，你忘了我刚才的话了吗？今日矿脉中的萧家人，一个都别想活命。萧家人中，自然也包括你。”
<br/>　　“你们原本就想杀我...”五长老愤怒至极。
<br/>　　“哈哈哈哈。”慕容槐道，“称霸紫云城，有我们慕容家就行了，你们萧家，只能成为我们脚下的尸体。”
<br/>　　“父亲。”一旁的萧若寒惊叫了一声，立刻打算救援，却被慕容槐一掌震得重伤。
<br/>　　“臭小子。”慕容槐冷笑道，“真以为自己算什么天才了。这些年，借着你爱慕娇儿小姐，才能对你多番利用，现在你已经没用了，死吧。”
<br/>　　“你...你们骗我。”萧若寒惊怒交加。
<br/>　　“嘭、嘭。”慕容槐打出了两掌，一掌击毙了萧若寒，一掌杀了五长老。
<br/>　　明显可以看到，两人的尸体上，眼睛瞪得极大，显然是死不瞑目。
<br/>　　萧逸看了他们的尸体一眼，但没有丝毫怜悯。
<br/>　　一个被权力蒙蔽了眼睛，早已丧失理智，心中毫无情义，甚至不惜勾结外人残害同族；
<br/>　　一个中了美人计，为了一个阴险的女子，竟不惜一切，乃至罔顾家族安危，向外人摇尾讨好。
<br/>　　这样的两个人，就算死了，萧逸也不会同情可怜他们。
<br/>　　不过，他们的死，倒也为萧逸争取了不少时间。
<br/>　　这时，慕容槐看着五长老没有闭上的眼睛，竟得意扬扬，“两个蠢货，既然你死了，我也不怕告诉你，我们家族已经行动了，你们整个萧家都会被灭掉。”
<br/>　　“你什么意思？”四长老大喝一声。
<br/>　　慕容槐抬起了头，一脚踢飞五长老的尸体，道，“还不明白吗？我们的目的本来就不单止萧家矿脉，而是你们整个萧家。”
<br/>　　“矿脉这边的武者力量，起码占了你们萧家三成的武者力量。而我们慕容家矿脉的武者，早就回到家族中汇合了，马上就会对你们萧家发起强攻。”
<br/>　　“萧家少了你们这三成力量，根本挡不住我们慕容家。等我将你们这里的人都杀个干净，会立刻返回紫云城，你们萧家一下子少了三个长老，注定会灭亡。”
<br/>　　慕容槐说罢，慕容蝎讥讽道，“当然了，这也是因为你们萧家的这两个蠢货，我们的计划才这么顺利。”
<br/>　　这时，嗖..嗖..嗖....
<br/>　　一连窜破空声袭来，八道人影忽然出现，并站在了慕容槐和慕容蝎身后。
<br/>　　这八人，全是慕容家的执事。其中三人还是精英执事，一个后天九重，两个后天八重。
<br/>　　他们跟随慕容槐和慕容蝎而来，一直潜伏在外面。
<br/>　　慕容槐大手一挥，冷喝道，“给我血洗萧家矿脉，一个不留。”
<br/>　　“是。”八人应答一声，当即出手。
<br/>　　恰在此时，一道霸道的冷喝声出忽然出现，“噬火百刃。”
<br/>　　一道火刃当即飞出，将一个执事当场击杀。
<br/>　　“嗯？萧逸？”慕容蝎皱起了眉头。
<br/>　　不错，出手的是萧逸。
<br/>　　“哼。”萧逸冷哼一声，手中凝聚出一把火剑，指向慕容家执事，冷笑道，“慕容家的执事，交给我了。”
<br/>　　“慕容槐和慕容蝎交给我们。”四六长老忽然大喝一声，并瞬间出手。
<br/>　　“你们不是被黑墨曼陀罗压制了真气吗？怎么可能出手？”慕容槐脸色大变。
<br/>　　谁都没有注意到，在四六长老背后，各自有一个无形漩涡，已将他们体内的黑墨曼陀罗武魂力量吸收殆尽，让他们恢复了实力。
<br/>　　萧逸大喝道，“今日我倒要看看，你们有什么本事血洗我萧家矿脉。”
<br/>　　话音刚落，三人瞬间出手。
