        &lt;h3&gt;第三卷第一百六十五章  刘诚之的想法&lt;/h3&gt;
<br/>
<br/>        董总和凌辰听见刘诚之自告奋勇的要去甸城，都看着刘诚之，齐声问:“你要去？为什么”
<br/>
<br/>        “我是一个名不见经传的小人物，过去看看，他们也不会很在意的，有可能敷衍敷衍我就是，我游游山玩玩水，顺便蹭吃蹭喝，说不定还能拿点外快，比凌总给我发的工资强多了。”刘诚之吊儿郎当的说。
<br/>
<br/>        “刘诚之，甸城也不是你想象的那么简单，你别逞能了。”董亚子不同意刘诚之的方案。
<br/>
<br/>        “那你过去？羊入虎口，投怀送抱？”刘诚之看了一眼董亚子。
<br/>
<br/>        “就这样决定了吧，凌总和董总年后先清查别的地方，甸城我过去！”刘诚之怼完董亚子直接就霸气的做了决定。
<br/>
<br/>        “诚之啊，我代表我们所有的工作人员感谢你！”董总站起来拿起一杯酒给刘诚之敬酒。
<br/>
<br/>        凌辰也站起来了，他都没有想到刘诚之会有这么样的担当。
<br/>
<br/>        他轻轻松松的说着玩笑话，其实是做了一个决定，将危险包揽到了他自己的身上。
<br/>
<br/>        刘诚之去甸城的话，按照何少华的性格绝对是看不上刘诚之，确实是会敷衍刘诚之的，也会放松对刘诚之的警惕，如果他和董亚子过去就不一样了。
<br/>
<br/>        “诚之，谢谢你为我分担！”凌辰也拿起了酒杯。
<br/>
<br/>        “我是为了我自己呢，我也想干一番事业，哈哈哈。”刘诚之笑哈哈的说，完全没有了那种屁颠屁颠的跟在凌辰后面跑路的样子。
<br/>
<br/>        董亚子哼了一声，狠狠的瞪了一眼刘诚之。
<br/>
<br/>        从董总家里出来，阴历十二月份的夜晚，蓝墨色的天空，繁星点点，俩个人都喝了点酒，看着满天繁星，凌辰对刘诚之说：
<br/>
<br/>        “诚之，你在我身边呆了不到两年，你这样做让我有些不适应。”
<br/>
<br/>        “我不是为了你，我是为了董亚子，董亚子虽然霸道一点，但是呢还是个可爱的人，她不应该为了家族而牺牲她自己。”
<br/>
<br/>        “你是喜欢董亚子吗？”
<br/>
<br/>        “错了，董亚子喜欢你，不过落花有情，流水无意而已，你的心里有着小雅姐姐的，要不然你和董亚子也是不错的一对。你回去之后去看看小雅姐姐，如果她真的有男朋友了，你就答应了董亚子，我和董亚子之间是误会，那一天是她拽着我的耳朵不小心俩个人碰上了。”
<br/>
<br/>        “诚之！”
<br/>
<br/>        “凌大哥，这两年在你身边你让我学会了很多，真的很感谢你，我去甸城也是想看看我有没有独当一面的能力，所以你不用感动，再说了，作为公司的一员我也是应该为公司分担的。”
<br/>
<br/>        “哈哈哈，我真的是很高兴，能让刘公子帮我分担，我凌辰上辈子肯定是做了好事情了。”
<br/>
<br/>        “凌大哥，不好意思啊，无论是什么样的人总是需要工作的吧。谢谢你收留我，让我学会了很多。”
<br/>
<br/>        “这样吧，回去就任命你为公司副总吧。”
<br/>
<br/>        “别，你就当我是你身边跑腿的，这样我做事情更方便。”刘诚之心里有他的计划和思虑。
<br/>
<br/>        “何少华也认识你了。我怕.......”
<br/>
<br/>        “何少华最大的优点就是看人下菜，他不会对我这个跑腿的注意的，如果是注意的话，他也是要拉拢我。在这次来甸城的时候他没有拿到总代，但是董总也说了，他已经向董家提亲了，他会认为董家一定会答应他的。总代没有拿到，那么势必会要娶到董亚子。”刘诚之分析着，如果董亚子嫁给凌辰，他还感觉蛮好的，要是嫁给何少华，他就感觉吞了一只苍蝇一样。
<br/>
<br/>        奇了怪了，董亚子嫁给谁关他什么事情？难道就因为那个first  kiss？
<br/>
<br/>        其实他在学校的时候是很少说话的，他家的家庭教育很好，对人礼貌有加，而且心思细腻，会顾虑到别人的感受，他给人的感觉就是那种温文有礼，淡淡的而且有点疏远的感觉。
<br/>
<br/>        他从来都不会很主动的去迎合一个人，而且在他的感情理念里就是那种一生一世一双人，情深意长，在没有碰到让自己动心的女孩子是不会谈情说爱的。
<br/>
<br/>        研究生毕业，原本是要回去上班，但是他还想闯荡一番，所以他毕业之后就和家里人商量了一下，先出来跑跑世界，学习学习外面的管理经验，锻炼锻炼处事的方法。
<br/>
<br/>        凌辰虽然性子冷情，但是做事情还是不错的，在凌辰的身边，他学到了很多学校里没有学到的知识，见到了各色各样的人。
<br/>
<br/>        社会真的是很能教会人做人。
<br/>
<br/>        快两年的时间，刘诚之改变了很多，再也不是在校的研究生了，但是这也离不开他强大的家庭环境教育和他聪慧的头脑。
<br/>
<br/>        “其实，对你来说都无关紧要不是吗？”凌辰看着满天的星辰。
<br/>
<br/>        “是无关紧要啊，我说了我现在还是公司的一员，我也想出去真正的试试我的能力。有机会为什么我不抓住呢？”
<br/>
<br/>        “董总那天的暗示已经说明了甸城有问题，原本不是我们代理商该负的责任，打击假冒伪劣产品是厂家和有关部门来做的，但是，我因为拿到了总代，我必须要和厂家站在一条线上，责任和担当都应该是我的啊！”凌辰有些忧心。
<br/>
<br/>        “放心吧，我只是过去给你探探路，以后的事情还是你来做。”
<br/>
<br/>        夜，有些冷冽，俩个人边走边聊。
<br/>
<br/>        “我这次回去，把公司和家里的事情处理一下，再去北京看看宁雅，如果宁雅真有男朋友了就祝她幸福，以后就做一个同学，如果没有男朋友，我就和她敞开了心怀，看看她是什么意思。我不想后悔一生，还是想让自己明白一点。”凌辰此时将刘诚之当一个可以倾诉的朋友了。
<br/>
<br/>        “你早都应该这样了，前一次去陈子昂公司的时候你们就应该找时间聊一聊，还有，要是喜欢一个人我觉得应该对她言明，暖昧似乎不妥。”刘诚之没有恋爱经验，他觉得他们已经从同学的状态转化成恋人的状态了，是不是就应该让对方明白自己的心意？
<br/>
<br/>        “哈哈哈，没有想到你比我小倒是比我有经验。”
<br/>
<br/>        “我不是比你有经验，我只是旁观者清而已，人，大概都是看不清楚自己吧！”
<br/>
<br/>        “对了，陈子昂那次打电话给我，让我把公司的转账信息给她，宁雅也给我打了一次电话，因为忙，我就没有给他们转账信息，哪一天你和陈子昂联系一下吧。”
<br/>
<br/>        “哈哈哈，你啊，人家给你还钱的时候，你不要，现在反而让我去问人家要钱。”
<br/>
<br/>        “那个时候不是心情不好嘛！”
<br/>
<br/>        “年后吧，这几天不适合要钱的。”刘诚之心里嘀咕着，你是老板，你愿意怎么样就怎么样！
