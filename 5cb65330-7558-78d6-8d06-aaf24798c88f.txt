　　第671章
<br/>　　犬之忠诚，鹿之谨慎，刀之锋利，这便是崔白玉身边三名丫鬟的性格。
<br/>　　有她们三人相助，崔白玉做起事来会更顺手。当然，也仅仅是顺手而已，崔白玉并不会觉得她们能给自己提供更大帮助，这三人也不会妄自菲薄觉得自己是崔白玉不可或缺的。
<br/>　　诸多事情，还需要崔白玉自己分析，自己做出决断。事实上，她并不信任她们，她也不试图去信任任何人，她能信任的只有自己。
<br/>　　在房间里，她坐在案后，桌上堆满关于程大雷的各种资料。房间整齐而有干净，崔白玉办事的时候不喜欢任何打扰。小鹿三人进出房间时，尽量不发出任何声音。
<br/>　　她们尽量不走进房间。
<br/>　　所以房间内只有崔白玉一个人，她阅读着桌上的资料，想要了解关于程大雷的全部。
<br/>　　桌上是堆积书山的资料，程大雷恐怕都不敢相信，关于自己的资料竟能到这种地步。事实上，相府很早就在关注程大雷，将他定位有危险性的人物之一。
<br/>　　能够在诸多次博弈中立于不败之地的势力，相府有自己专业的组织架构，并非程大雷这种草台班子可比。
<br/>　　在相府内部，有人专门收集情报，有人负责训练杀手，同样也有人根据收集到的情报，做出判断。
<br/>　　崔白玉就是这种人。
<br/>　　她桌上放着一杯清茶，逐字翻阅着手中的资料，翻完后，便去换另外一份。在阅读这些资料的时候，她尽力让自己保持平静，不带有任何情绪与偏见。
<br/>　　并不会因程大雷出身是个山贼便如何如何，她若枯坐的老僧，试图通过这些资料，在心中描绘出程大雷的模样。
<br/>　　只有真正了解自己的敌人，方才能在战争之前获得更多胜算。战争，终究是人与人之间进行的。
<br/>　　差不多同样时间，程大雷也窝在自己书房内，翻阅关于崔白玉的情报。这自然是一项无聊的工作，小蝶又份外跳脱些，过了没大会儿，程大雷便与小蝶扯起了闲篇。
<br/>　　而崔白玉仍专注在纸堆中，在脑海中勾勒程大雷的模样。
<br/>　　相府有一份程大雷的画像，是一名画师根据程大雷在长安时的相貌所绘。那时候，他正冒充牛三斤，穿着简陋的盔甲，目光猥琐又狡诈。那时候，他还年轻，模样当然丑陋得很。
<br/>　　崔白玉盯着这幅画像，静坐不动，脑海中思索半晌，手指轻轻叩着桌面。
<br/>　　他，究竟是一个怎样的人？
<br/>　　从青牛山起家，由长安城到凉州，生生被其撕扯出一片天地。天下诸人皆想除之而后快，但皆没能做到，不仅如此，他反而活得越来越好。
<br/>　　崔白玉忽地皱起眉头，眉目中闪过一抹厉色，她突然察觉到：有些事情不对。
<br/>　　自从他在青牛山出世，手下不多时便聚集一批精兵强将，能抵挡戎族的十万大军，又可在凉州扯起一支人马。
<br/>　　崔白玉隐隐觉得，似乎有某种自己所不知道的力量，在暗中帮助程大雷。
<br/>　　这只是一种直觉，忽然从心里冒出来，可当她顺着这个念头想下去时，便觉得不寒而栗。
<br/>　　程大雷出世以来，刚开始武艺并不如何，但没有多长时间，便可与天下一流高手过招，他的武艺从何而来……
<br/>　　火药、辣椒，青牛山那场连绵大雨……这些东西就像是突然冒出来的相仿，那么他究竟从何处得来。
<br/>　　如果一开始只是推测，那么崔白玉现在几乎已经能够确定，有某种力量在暗中帮助程大雷。只是，她现在还想不清楚，是谁在帮助程大雷。
<br/>　　将军府、其它诸侯王，或者是那高高在上的帝国天子。
<br/>　　想到这里，崔白玉忽然想到哪位已经去世的明帝。据说，他和程大雷的关系不错，在长安时，就是明帝法外开恩放了程大雷一马。而程大雷和明帝最宠爱的一子一女李乐天与李婉儿都有极好的关系，他们二人没少得到程大雷的帮助。
<br/>　　莫非，程大雷本就是明帝早早埋下的一颗钉子。
<br/>　　越想这个可能性越大，关于那位逝去的明帝，外人都经常用无能二字评价他。事实上，崔白玉要比普通人了解得更多。的确，在他的任内，戎族两次侵犯帝国，两次与戎族耻辱和谈。
<br/>　　这是他逃不过的千古骂名，但反过来想，能在这种情况下，还撑着帝国三十年基业不乱，已经是极不容易的事情。否则的话，三十年前，帝国就已经是乱世了。
<br/>　　如果程大雷是明帝事先埋下的钉子，为的就是给苟延残喘的帝国续命，这个可能性还是极大的。当真如此，那么自己还是把这场战争想的太简单了，自己要面对的敌人，可不仅仅是程大雷一人。
<br/>　　那位如今身在长安的天子，怕也不会太过安分。
<br/>　　不安常理出牌，总有出人预料之举，手下兵将极强的战斗力，以及那些层出不穷的武器和策略。
<br/>　　崔白玉皱起眉头，感觉到自己所面对的是一个棘手的敌人。最重要的一点，是到现在她还不知程大雷是一个怎样的人。
<br/>　　现在她面前摆着几首诗，都是出自程大雷手中。崔白玉已经看了一遍又一遍，心中的困惑仍旧挥之不去。
<br/>　　诗词自然都是好的，但总觉得不该出自程大雷，倒不是因为他是个山贼便小看他，而是诗词中的胸怀抱负，与程大雷平时做人做事完全不同。
<br/>　　诗词是由心而发，他没有这个心，便不能有这样的诗词。
<br/>　　而如今的问题是，他偏偏有了，这才是真正令崔白玉想不通的问题。
<br/>　　不过，崔白玉也并不着急，今天仅仅是个开始，她还有十天的时间，去琢磨程大雷是个怎样的人。
<br/>　　崔白玉相信，在十天后，自己总能得到答案。从心里彻底了解程大雷，从而制定出必胜的策略。
<br/>　　夜深孤灯慢翻书。
<br/>　　崔白玉总是有个很有耐心的人，喜欢收集资料，分析资料背后的人，从而了解一个人。
<br/>　　差不多同样时间，程大雷窝在书房里，也面对一个困难。
<br/>　　今晚自己该去苏樱房中歇，还是去樊梨花房中。
<br/>(本章完)
